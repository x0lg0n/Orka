# Partial Release

Release funds in stages as work progresses.

## When to Use Partial Release

- Long-running milestones
- Phased deliverables
- Progress-based payments
- Risk mitigation

## Partial Release Process

1. Specify release amount
2. Document progress made
3. Client approves partial release
4. Funds transferred
5. Remaining balance stays in escrow

## Tracking Partial Releases

- View release history
- Track remaining balance
- Document each release reason
