# Release Funds

Release funds to freelancer when work is approved.

## Release Conditions

- Milestone marked as complete
- Client approves deliverables
- No pending disputes
- Sufficient balance in escrow

## Release Process

1. Client clicks **Approve Milestone**
2. Funds released from escrow
3. Transferred to freelancer's wallet
4. Transaction recorded on Stellar
5. Both parties receive confirmation

## Timing

- Immediate release on approval
- Funds available in freelancer's wallet within seconds
- Transaction visible on Stellar explorer
