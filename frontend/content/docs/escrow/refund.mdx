# Refund

Request refunds when projects don't proceed as planned.

## Refund Scenarios

- Project cancelled
- Scope changed significantly
- Mutual agreement to terminate
- Dispute resolution

## Refund Process

1. Initiate refund request
2. Provide reason for refund
3. Other party reviews request
4. Agreement reached
5. Funds returned to original wallet

## Refund Policies

- Client-initiated refunds: immediate
- Mutual refunds: require both approvals
- Dispute refunds: mediated through ORKA support
