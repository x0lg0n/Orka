# Escrow

ORKA uses Stellar blockchain for secure milestone-based escrow payments.

## How Escrow Works

1. Client funds escrow for milestone
2. Funds held securely on Stellar
3. Freelancer completes work
4. Client approves milestone
5. Funds released to freelancer

## Benefits

- **Security** - Funds protected on blockchain
- **Transparency** - All transactions visible
- **Automation** - Smart contract enforcement
- **Low fees** - Stellar's minimal costs
