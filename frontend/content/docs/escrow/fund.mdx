# Fund Escrow

Fund escrow to secure milestone payments.

## Funding Process

1. Go to project milestones
2. Select milestone to fund
3. Click **Fund Escrow**
4. Choose funding amount
5. Confirm transaction in Freighter

## Funding Options

- **Full amount** - Fund entire milestone
- **Partial** - Fund portion now
- **Multiple milestones** - Fund several at once

## Transaction Details

- Stellar network fee: ~0.00001 XLM
- Confirmation time: 3-5 seconds
- Transaction record on blockchain
