# Escrow Security

ORKA uses Stellar blockchain for secure, transparent escrow.

## Security Features

- **Blockchain storage** - Funds on Stellar network
- **Smart contracts** - Automated enforcement
- **Multi-signature** - Required approvals
- **Audit trail** - Complete transaction history

## Stellar Security

- Decentralized network
- Cryptographic protection
- Immutable transaction records
- Fast confirmation times

## Best Practices

- Always verify wallet addresses
- Review transactions before signing
- Keep Freighter extension updated
- Report suspicious activity
