# Stellar Payments

ORKA uses Stellar for fast, secure, low-cost transactions.

## Why Stellar?

- **Speed** - 3-5 second confirmation
- **Cost** - Fractions of a cent per transaction
- **Security** - Decentralized network
- **Scalability** - Handles thousands of transactions

## Stellar Basics

- Native currency: XLM (lumens)
- Assets: Tokenized currencies
- Wallets: Freighter browser extension
- Explorers: View transactions on-chain

## Getting Started

1. Install Freighter wallet
2. Fund wallet with XLM
3. Connect to ORKA
4. Start making payments
