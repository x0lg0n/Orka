# Payments

ORKA handles all payments through Stellar blockchain for security and transparency.

## Payment Features

- **Stellar transactions** - Fast, low-cost transfers
- **Currency conversion** - Multi-currency support
- **Transaction history** - Complete payment records
- **Failed payment handling** - Automatic retry system

## Payment Types

- Escrow funding
- Milestone releases
- Invoice payments
- Refunds
