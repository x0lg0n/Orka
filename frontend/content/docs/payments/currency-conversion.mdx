# Currency Conversion

ORKA supports multi-currency transactions with automatic conversion.

## Supported Currencies

- XLM (Stellar Lumens)
- USD (US Dollar)
- EUR (Euro)
- GBP (British Pound)
- And more

## How Conversion Works

1. Select payment currency
2. ORKA shows conversion rate
3. Confirm transaction
4. Funds converted automatically
5. Recipient receives in their currency

## Conversion Rates

- Real-time market rates
- Transparent fee structure
- No hidden charges
- Rate locked at transaction time
