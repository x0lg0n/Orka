# Failed Payments

ORKA helps you resolve payment issues quickly.

## Common Failure Reasons

- Insufficient funds
- Network congestion
- Invalid wallet address
- Transaction timeout

## Resolution Process

1. ORKA detects failure
2. Notification sent to user
3. Error details provided
4. Retry option available
5. Support escalation if needed

## Prevention Tips

- Maintain sufficient wallet balance
- Verify wallet addresses
- Check network status
- Update Freighter extension
