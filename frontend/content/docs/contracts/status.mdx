# Contract Status

Monitor contracts through their lifecycle.

## Status Types

- **Draft** - Being created
- **Pending** - Sent for approval
- **Signed** - Executed by all parties
- **Active** - Currently in effect
- **Expired** - Past end date
- **Terminated** - Ended early

## Status Tracking

- Real-time status updates
- Email notifications
- Dashboard overview
- Filter and search contracts

## Contract Actions

- Renew expiring contracts
- Terminate early
- Request amendments
- Archive completed contracts
