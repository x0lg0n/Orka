# Contract Templates

Create templates to streamline contract creation.

## Creating Templates

1. Create a contract
2. Click **Save as Template**
3. Name your template
4. Add template variables
5. Save for future use

## Template Variables

- {{client_name}}
- {{project_name}}
- {{start_date}}
- {{total_amount}}
- {{payment_terms}}

## Managing Templates

- Edit existing templates
- Delete unused templates
- Share with team members
- Import/export templates
