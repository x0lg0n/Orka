# Troubleshooting

Solve common Freighter problems.

## Common Issues

- **Extension not showing** - Check installation
- **Connection failed** - Refresh and retry
- **Transaction timeout** - Check network status
- **Balance not showing** - Sync wallet

## Solutions

1. Refresh browser
2. Reinstall extension
3. Clear browser cache
4. Check internet connection
5. Update extension
