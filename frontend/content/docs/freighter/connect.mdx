# Connect to ORKA

Connect your wallet to start using ORKA's payment features.

## Connection Process

1. Open ORKA settings
2. Navigate to **Wallets**
3. Click **Connect Freighter**
4. Approve connection in popup
5. Wallet address displayed

## What ORKA Can Do

- Request transaction signatures
- View wallet balance
- Initiate payments
- Check transaction status

## What ORKA Cannot Do

- Access private keys
- Sign without approval
- Modify wallet settings
- Access other wallets
