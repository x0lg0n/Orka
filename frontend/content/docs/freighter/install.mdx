# Install Freighter

Get started with Freighter in minutes.

## Installation Steps

1. Visit [freighter.app](https://freighter.app)
2. Click "Install for Chrome"
3. Add extension to browser
4. Pin to toolbar for easy access

## First-Time Setup

1. Click Freighter icon
2. Create new wallet or import existing
3. Save recovery phrase securely
4. Set password for extension

## System Requirements

- Chrome or Firefox browser
- Internet connection
- Stable browser version
