# Sign Transactions

Approve payments and other transactions securely.

## Transaction Flow

1. ORKA initiates transaction
2. Freighter popup appears
3. Review transaction details
4. Approve or reject
5. Transaction submitted to Stellar

## Security Checks

- Verify recipient address
- Check transaction amount
- Review network fees
- Confirm memo if included

## Best Practices

- Always review before signing
- Never share your password
- Report suspicious transactions
- Keep extension updated
