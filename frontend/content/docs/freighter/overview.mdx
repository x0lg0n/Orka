# Freighter

Freighter is a Stellar wallet browser extension for managing payments.

## What is Freighter?

- Browser extension for Chrome/Firefox
- Manages Stellar wallet
- Signs transactions
- Stores private keys securely

## ORKA Integration

- Connect wallet for escrow
- Sign payment transactions
- View transaction history
- Manage multiple wallets
