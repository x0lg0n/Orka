Freighter is a Stellar wallet browser extension that lets you connect your own wallet to ORKA for self-custody mode.

## What is Self-Custody?

In self-custody mode, you hold your own keys. ORKA never has access to your wallet. You sign transactions directly in your browser.

## Connecting Freighter

1. Install the Freighter browser extension
2. Create or import a Stellar wallet
3. Go to ORKA settings → **Wallet**
4. Click **Connect Freighter**
5. Approve the connection in the extension

<Callout type="warning">
Make sure you back up your Freighter seed phrase. If you lose it, you lose access to your wallet permanently.
</Callout>

## Self-Custody vs Managed

| Feature | Managed Mode | Self-Custody |
|---------|-------------|--------------|
| Key storage | ORKA holds keys | You hold keys |
| Transaction signing | ORKA signs | You sign in browser |
| Recovery | Password reset | Seed phrase only |
| Best for | Most users | Crypto-native users |

## Signing Transactions

When you connect Freighter:

1. ORKA prepares the transaction
2. Freighter prompts you to sign
3. You review and approve in the extension
4. The transaction is submitted to Stellar

<Callout type="tip">
Self-custody gives you full control but requires more responsibility. Choose managed mode if you're not comfortable managing crypto keys.
</Callout>
