Milestones break your project into manageable chunks with clear deliverables and pricing.

## Creating Milestones

1. Open your project
2. Click **Add Milestone**
3. Enter the milestone name, description, and price
4. Set the deadline and deliverables

## Milestone Structure

Each milestone includes:

- **Name**: Brief title (e.g., "Discovery Phase")
- **Description**: Detailed scope of work
- **Price**: Amount for this milestone
- **Deadline**: When it's due
- **Deliverables**: What the client expects

## Tracking Progress

Track milestone status through its lifecycle:

1. **Draft**: Being created
2. **Active**: Work in progress
3. **Submitted**: Deliverables submitted for review
4. **Approved**: Client approved the work
5. **Released**: Payment released to freelancer

## Files and Evidence

Attach files to milestones:

- GitHub pull requests
- Figma design files
- Google Docs links
- Direct file uploads

<Callout type="tip">
When submitting work, attach as much evidence as possible. ORKA's AI verification checks these links to confirm delivery.
</Callout>
