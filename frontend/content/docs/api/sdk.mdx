# SDK

ORKA provides SDKs for popular programming languages.

## Available SDKs

- JavaScript/TypeScript
- Python
- Ruby
- PHP

## Installation

```bash
npm install @orka/sdk
```

## Quick Start

```javascript
import { Orka } from '@orka/sdk';

const orka = new Orka({ apiKey: 'your-key' });
const projects = await orka.projects.list();
```
