# Create Invoices

Create invoices manually or from milestones.

## Manual Creation

1. Go to **Invoices**
2. Click **New Invoice**
3. Add line items
4. Set payment terms
5. Send to client

## From Milestones

1. Complete milestone
2. Click **Create Invoice**
3. Auto-populated details
4. Review and send

## Invoice Details

- Invoice number (auto-generated)
- Client information
- Line items with amounts
- Tax calculations
- Payment due date
