# Invoices

Generate professional invoices and track payments seamlessly.

## Invoice Features

- **Auto-generation** - Create from milestones
- **Custom templates** - Brand your invoices
- **Payment tracking** - Monitor payment status
- **Export options** - PDF and CSV formats

## Invoice Lifecycle

1. **Create** - Generate invoice
2. **Send** - Deliver to client
3. **Track** - Monitor payment status
4. **Receive** - Payment processed
5. **Reconcile** - Update accounting records
