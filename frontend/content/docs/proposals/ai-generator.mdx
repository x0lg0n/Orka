# AI Generator

ORKA's AI helps you create compelling proposals in minutes.

## How It Works

1. Enter project details
2. Describe your services
3. AI generates proposal content
4. Review and customize
5. Send to client

## Tips for Best Results

- Be specific about project scope
- Include budget expectations
- Mention timeline requirements
- Add relevant portfolio examples

## Customizing AI Output

- Edit generated text
- Add your branding
- Include custom sections
- Adjust pricing details
