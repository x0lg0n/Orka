# Proposal Versions

Track changes and maintain proposal history.

## Version Control

- Automatic versioning on edits
- Compare different versions
- Restore previous versions
- View change history

## When to Create Versions

- Major content changes
- Pricing adjustments
- Scope modifications
- Client feedback incorporation

## Managing Versions

1. Go to proposal settings
2. Click **Version History**
3. Select version to view
4. Restore or compare as needed
