# Proposal Status

Monitor your proposals throughout their lifecycle.

## Status Types

- **Draft** - Not yet sent
- **Sent** - Delivered to client
- **Viewed** - Client has opened
- **Accepted** - Client approved
- **Rejected** - Client declined
- **Expired** - Past deadline

## Tracking Features

- View history of changes
- See when client opens proposal
- Track time spent reviewing
- Get approval notifications

## Analytics

- Open rates
- Review time
- Conversion tracking
