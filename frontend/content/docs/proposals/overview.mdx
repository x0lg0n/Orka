# Proposals

Create professional proposals to win new business. ORKA helps you craft compelling proposals with AI assistance.

## Proposal Features

- **AI Generator** - Create proposals with AI
- **Templates** - Save and reuse proposal formats
- **E-signatures** - Get approvals digitally
- **Tracking** - Know when clients view proposals

## Creating a Proposal

1. Go to **Projects** or **Clients**
2. Click **New Proposal**
3. Choose a template or start from scratch
4. Add sections and details
5. Send to client
