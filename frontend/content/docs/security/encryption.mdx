# Encryption

Your data is protected with industry-standard encryption.

## Encryption Methods

- **At rest** - AES-256 encryption
- **In transit** - TLS 1.3
- **Database** - Encrypted storage
- **Backups** - Encrypted backups

## Key Management

- Hardware security modules
- Regular key rotation
- Secure key storage
- Access controls

## What's Encrypted

- Personal information
- Financial data
- Documents
- API credentials
