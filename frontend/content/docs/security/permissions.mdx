# Permissions

ORKA uses role-based access control.

## Permission Levels

- **Owner** - Full access, billing
- **Admin** - Manage team and projects
- **Member** - Edit assigned work
- **Viewer** - Read-only access

## Setting Permissions

1. Go to workspace settings
2. Click **Team Members**
3. Edit member role
4. Save changes

## Custom Permissions

- Project-level access
- Client-specific permissions
- Feature restrictions
- Time-based access
