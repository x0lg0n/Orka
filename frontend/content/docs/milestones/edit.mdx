# Edit Milestones

Update milestones when project requirements evolve.

## What You Can Edit

- Title and description
- Deadline and timeline
- Payment amount
- Deliverable details
- Dependencies

## Editing Process

1. Go to project milestones
2. Click on milestone to edit
3. Make changes
4. Save updates
5. Notify team of changes

## Impact of Changes

- Timeline adjustments affect project schedule
- Budget changes require escrow updates
- Scope changes may need client approval
