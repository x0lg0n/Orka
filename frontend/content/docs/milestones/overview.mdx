# Milestones

Divide projects into clear milestones for better tracking and payments.

## Milestone Features

- **Clear deliverables** - Define specific outputs
- **Timeline tracking** - Set and monitor deadlines
- **Payment gates** - Link payments to completion
- **Approval workflow** - Client sign-off process

## Creating Milestones

1. Go to project settings
2. Click **Milestones**
3. Add new milestone
4. Set deliverables and deadline
5. Link to escrow payment
