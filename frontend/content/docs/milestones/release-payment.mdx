# Release Payment

Release funds from escrow upon milestone completion.

## Release Process

1. Milestone approved by client
2. Payment release initiated
3. Funds transferred to freelancer
4. Transaction recorded
5. Both parties notified

## Payment Details

- Automatic calculation of fees
- Transaction history updated
- Tax documentation generated
- Receipt provided to both parties

## Troubleshooting

- Verify wallet connection
- Check escrow balance
- Contact support if issues arise
