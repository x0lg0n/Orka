## Questions & help

- **Bugs and feature requests:** open an
  [issue](https://github.com/x0lg0n/Orka/issues) using the templates.
- **General questions:** open an issue or a GitHub discussion.

## Security

Please **do not** open public issues for security vulnerabilities. Follow the
[Security Policy](./SECURITY.md) for private, responsible disclosure.

## Before you file

- Search existing [issues](https://github.com/x0lg0n/Orka/issues) — your question
  may already be answered.
- Include your environment (OS, Node/pnpm/Rust versions, network, custody mode)
  and clear reproduction steps.
