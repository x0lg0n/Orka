      - `export interface ReleaseArgs { contractId: string; milestoneId: number; }`

      - `export function createOrkaClient(opts: OrkaClientOptions)` returning `{ fundEscrow, releaseMilestone, getContractState }`.
      - `fundEscrow(args: FundArgs): Promise<FundResult>` → `POST {baseUrl}/escrow/fund` with `{contractId, milestoneIds, mode: opts.mode}`. On Mode A response `{txHash}`; on Mode B response `{txXdr}`. Return accordingly.
